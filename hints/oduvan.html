Qual pode ser o primeiro passo para resolver esse problema?

A maneira mais simples de resolver esse problema é utilizando comparadores.

O que são comparadores?

Comparadores são operadores que permitem comparar dois valores em Python. Eles são usados para testar se uma determinada condição é verdadeira ou falsa. Os principais comparadores em Python são:
== - verifica se dois valores são iguais;
!= - verifica se dois valores são diferentes;
< - verifica se um valor é menor que outro;
> - verifica se um valor é maior que outro;
<= - verifica se um valor é menor ou igual a outro;
>= - verifica se um valor é maior ou igual a outro.

O que faço depois de comparar?

Será necessário usar condicionais para obter uma saída diferente para cada entrada.

Por exemplo, se num > 0, precisamos escrever positivo no compilador.

E como posso escrever essa condicional?

Um exemplo pode ser um código utilizado para determinar se uma pessoa é menor de idade, adulta ou idosa. 

if idade < 18:
    print("Você é menor de idade.")
elif idade >= 18 and idade <= 65:
    print("Você é adulto.")
else:
    print("Você é idoso.")
